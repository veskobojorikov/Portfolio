Cinema Hub

Cinema Hub е уеб приложение за представяне и организиране на филмово съдържание. Проектът е фокусиран върху модерен UI и лесна навигация.

Основната идея е потребителят да може да разглежда филми по категории и жанрове с ясна и подредена структура.

Функционалности

Сайтът включва каталог с филми, разделени по категории, с изображения и описания.

Интерфейсът е направен за бързо и удобно навигиране между различни секции.

Технологии

HTML, CSS, JavaScript. Концепцията е базирана на ASP.NET структура.

Цел на проекта

Демонстрация на умения в UI дизайн, frontend разработка и структуриране на уеб приложения.

Снимки

Технологии

HTML, CSS, JavaScript, Mariadb, Asp.Net Web App
